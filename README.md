## 🚀 Projetos em Destaque

### 🔄 [GitHub Pipeline ETL](https://github.com/avelinoandre)
Pipeline ETL que extrai dados de repositórios via **REST API do GitHub**, transforma as informações com **Pandas** em DataFrames e exporta para `.csv`. O carregamento é automatizado via API, criando repositórios de destino e realizando commits programáticos com **Python + Requests**.

`Python` `Pandas` `REST API` `ETL` `Requests`

---

### 🎮 [SteamView](https://github.com/avelinoandre)
Plataforma web para pesquisa e visualização de dados de jogos (preço, nota, rankings). Integra as APIs da **Steam** e **RAWG**, com backend em **Django**, banco de dados **PostgreSQL** e deploy na **Azure**. Desenvolvido com testes E2E (**Cypress**) e gestão ágil via **SCRUM + Jira**.

`Django` `PostgreSQL` `Azure` `Cypress` `Steam API` `RAWG API` `SCRUM`
